# 6-7 / 67 🤚

Um site de uma página sobre o meme do **67**. É parvo de propósito.

São 16 mini-coisas à volta do número 67 — um contador, uma bola de cristal que responde sempre "6… 7", uma calculadora em que tudo dá 67, um relógio preso nas 6:07, uma roda da sorte viciada, um pet virtual, uma fábrica idle, etc. Tem conquistas para desbloquear e uns easter eggs à mistura.

## Correr

Abre o `index.html`. Não precisa de servidor nem de instalar nada — duplo clique e está.

## Como foi feito

HTML, CSS e JavaScript, sem frameworks nem build. Os sons são todos gerados no browser (Web Audio para os tons/batida, Web Speech para a voz), por isso não há ficheiros de áudio.

São 3 ficheiros: `index.html`, `styles.css`, `script.js`.

Há um código secreto escondido — boa sorte. 👀
